✅ Task 3: Arrange the CI/CD Pipeline Flow
📌 Given Steps
Deploy Application
Write Code
Run Tests
Build Application
Monitor Application
🛠️ Correct Order
1. Write Code
2. Build Application
3. Run Tests
4. Deploy Application
5. Monitor Application
🎯 Explanation
Write Code → Developer writes application code
Build Application → Compile/package the code
Run Tests → Validate functionality
Deploy Application → Release to production/server
Monitor Application → Track logs, errors, and performance
